AVISO DE PRIVACIDAD PARA PROSPECTOS

DATOS PERSONALES RECABADOS POR RIFAS ENTRE AMIGOS GUADALAJARA

Para efectos de cumplir con las finalidades previstas en este Aviso de Privacidad, recabaremos de usted los siguientes
datos personales:

Datos de identificación
Datos de contacto

Le informamos que los datos personales anteriores los podemos obtener cuando usted nos los proporciona de forma
directa o bien de forma indirecta a través de fuentes de acceso público.

Le informamos que para efectos de cumplir con las finalidades previstas en el presente Aviso de Privacidad no daremos
tratamiento a datos personales sensibles.

USO DE DATOS PERSONALES

Finalidades primarias:

Los datos personales que usted nos proporcione podrán ser utilizados para las siguientes finalidades:

identificarle y registrarle como un posible prospecto de venta.
Determinar si usted es apto para adquirir nuestros productos o servicios.
Contactarle, a través de diversos medios de comunicación físicos y electrónicos con el propósito de ofrecerle nuestros
productos y servicios.
Dar seguimiento a sus solicitudes de información sobre nuestros productos y servicios o cualquier otra cuestión que
resulte de nuestra competencia.
Cumplir con obligaciones previstas en la normatividad aplicable así como cooperar con autoridades competentes en caso
de que así se requiera.

Finalidades secundarias:

De manera adicional, si usted no se opone, Rifas Entre Amigos Guadalajara mantendrá su información personal para las
siguientes finalidades:

Para posibles contactos posteriores.
Para enviarle información promocional sobre nuestros avances, eventos, promociones y servicios de Rifas Entre Amigos
Guadalajara así como, para mantener contacto con usted como posible participante.
Para realizar investigaciones de mercado y promoción de productos.
Para fines estadísticos.

MECANISMOS PARA EXPRESAR NEGATIVA AL TRATAMIENTO DE SUS DATOS PERSONALES

En caso de que no desee que sus datos personales sean tratados para alguna o todas las finalidades secundarias, desde
este momento nos puede comunicar lo anterior al whatsapp: +52 81 2647 5803.

La negativa para el uso de sus datos personales para fines adicionales o secundarios, no podrá ser un motivo para
negarle los servicios establecidos o dar por terminada la relación establecida con nosotros.

CON QUIÉN PODEMOS COMPARTIR SUS DATOS PERSONALES

Para cumplir con las finalidades exigidas legalmente o por autoridades competentes, Rifas Entre Amigos Guadalajara
sólo transferirá los datos necesarios en los casos legalmente previstos.

Tercero receptor de los datos personales: Autoridades competentes
Finalidad: En los casos legalmente previstos
Consentimiento: No necesario

procedimientos, requisitos y plazos para el ejercicio de sus Derechos ARCO y/o revocación del consentimiento

COOKIES

Rifas Entre Amigos Guadalajara utiliza varias tecnologías para mejorar la eficiencia de sus sitios web,
incluyendo su experiencia cuando navega por dichos sitios. Entre estas tecnologías se incluye el uso de
